# Getting started

This page exists to prove the template already supports MDX out of the box.

## First commands

```bash
bun install
bun run dev
bun run typecheck
```

## What you get

- Cloudflare adapter configuration
- Tailwind CSS v4
- Biome with pre-commit hooks
- Vitest and Playwright
- A sample JSON endpoint for quick smoke checks

## Next steps

1. Rename the project.
2. Replace the homepage with your product.
3. Add real Cloudflare bindings in `wrangler.jsonc`.
4. Regenerate Worker types with `bun run generate-types`.
